# alex-0d18-test-1

Static browser clock scaffold.

## Files

- `index.html`
- `styles.css`
- `clock.js`

## Run

Open `index.html` directly in a browser, or serve the repository root with any static file server.

For example:

```sh
python3 -m http.server 8080 --bind 0.0.0.0
```

Then visit `http://localhost:8080/`.

## Test

```sh
npm test
```
